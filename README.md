# AI Gym & Fitness Assistant

This project contains a frontend prototype and a backend API for an AI-powered fitness ecosystem.

## Frontend
- Location: `frontend/`
- Stack: React + Vite

## Backend
- Location: `backend/`
- Stack: FastAPI

## Run backend
```bash
cd backend
pip install -r requirements.txt
uvicorn app:app --reload --host 0.0.0.0 --port 8000
```

## Run frontend
```bash
cd frontend
npm install
node node_modules/vite/bin/vite.js
```
